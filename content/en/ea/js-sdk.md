## Installation

Add `@piebits/pc-js-sdk` dependency to your project:

<code-group>
  <code-block label="Yarn" active>

  ```bash
  yarn add @piebits/pc-js-sdk
  ```

  </code-block>
  <code-block label="NPM">

  ```bash
  npm install @piebits/pc-js-sdk
  ```

  </code-block>
</code-group>
